# 负数过过过

小游戏“负数过过过”，参考名侦探学院游戏规则。

教孩子学编程用版本，所以没有采用面向对象的写法，也没有拆分模块，并且选择字符界面库 [Rich](https://github.com/willmcgugan/rich)，简单友好。

## 运行方法

1. 安装运行环境，需要 python 3
2. 安装依赖

   ```shell
   pip install -r requirements.txt
   ```

3. 运行

   ```shell
   python game.py
   ```

## 开发环境

MacOS 11
VSCode 1.52
Python 3.9
